CSS Common Properties
CSS Tag name description
color:
 color the color property allows us to specify the text color of an HTML element.
font-size:
 Font Size The font-size property allows us to change the size of the font for any text based content.
background-color:
 Background Color The background-color property allows you to set the background color of an HTML element.
Height: & width:
 Height/Width The height and width properties specify the height and width of a HTML element respectively.
box-model:
 Box Model (Margin/Padding) The box model refers to the box layout of each HTML element.
  1. Content - The actual content (text, images) of the HTML element.
  2. Padding - Defined by the padding property. In-between the border and content of an element. Is transparent.
  3. Border - Defined by the border property. Surrounds the padding and content of an element.
  4. Margin - Defined by the margin property. Adds space between the border and other HTML elements. Is transparent.
border:
 Border The border property allows you set a border for your HTML element.
border-radius:
 Border Radius The border-radius property allows you to add rounded corners of a HTML elements.
